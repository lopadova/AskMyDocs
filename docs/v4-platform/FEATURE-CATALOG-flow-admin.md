# Feature Catalog: padosoft/laravel-flow-admin v1.0.0

**Purpose:** Professional Blade + Alpine admin panel for `padosoft/laravel-flow` — runs, approvals, outbox, definitions in one dark-first cockpit.

> Generated by background Explore agent during v4.2 cycle (2026-05-09). Use this catalog as the integration spec for sub-PR 6.

**Key Design Principles:**
- Deny-by-default authorizer (every mutation gated via `ActionAuthorizer`)
- Read-model adapter pattern: Eloquent for production, Array for deterministic E2E tests
- Public extension surface only (`Contracts/*`); internals stay `@internal`
- Pixel-perfect dark + light themes (cookie-persisted, per-user override)

## CONFIGURATION

| Feature | What it does | AskMyDocs benefit | Priority |
|---|---|---|---|
| **prefix** | URI prefix for all routes (default `flow`) | Mount at /admin/flows under existing AskMyDocs admin shell | SHOULD |
| **middleware** | Comma-separated stack (default `web,auth`) | Wire to `web,auth,can:viewFlowAdmin`; AskMyDocs Spatie role gate | MUST |
| **adapter** | `eloquent` (prod) or `array` (E2E seed-42) | Production: eloquent; Playwright spec uses `array` for reproducible screenshots | SHOULD |
| **authorizer** | FQCN implementing `ActionAuthorizer` (default `DenyAllAuthorizer`) | Bind a custom authorizer that maps Spatie roles → action permissions | MUST |
| **polling_interval_ms** | Auto-refresh cadence (default 4000ms) | Live updates without full reload; tunable for low-bandwidth ops | COULD |
| **theme_default** | `dark` or `light` (default `dark`) | Operator UX; persisted per-user via cookie | COULD |
| **step_viz_default** | `timeline`, `gantt`, or `dag` (default `timeline`) | Sane default for AskMyDocs flows; user can switch | COULD |

## AUTHORIZATION & MUTATIONS

| Mutation | Default Gate | AskMyDocs binding | Priority |
|---|---|---|---|
| **canResume(runId)** | `DenyAllAuthorizer` denies | Spatie role `flow.runs.resume` (super-admin + ops) | MUST |
| **canCancel(runId)** | denies | Spatie role `flow.runs.cancel` (super-admin only) | MUST |
| **canApprove(approvalId)** | denies | Spatie role `flow.approvals.act` (super-admin + dpo + curator) | MUST |
| **canReject(approvalId)** | denies | Spatie role `flow.approvals.act` | MUST |
| **canReplayWebhook(outboxId)** | denies | Spatie role `flow.outbox.replay` (super-admin + ops) | SHOULD |
