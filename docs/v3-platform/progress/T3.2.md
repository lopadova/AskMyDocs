### [06:18:00] Step 1: ConfidenceCalculator implementation
**Action:** Created `app/Services/Kb/Grounding/ConfidenceCalculator.php` implementing the formula from plan §2434:

```
confidence = 100 * (
    0.40 * mean_top_k_similarity +
    0.20 * threshold_margin +
    0.20 * chunk_diversity +
    0.20 * citation_density
)
```

Design choices (see L18):
- Weights are `private const WEIGHT_*` (not config) — they're contract, not knobs.
- End-of-method clamp `(int) round(max(0.0, min(100.0, $score)))` is load-bearing — schema has no CHECK constraint.
- Each sub-calc clamps to [0,1] via `clamp01()` for defense-in-depth.
- Boundary carve-outs:
  - 0 chunks → 0 (defense vs caller forgetting refusal short-circuit)
  - threshold === min_used → margin = 0
  - 0 answer words → density forced to 1 (refusal path neutrality)
  - threshold === 1.0 → margin = 0 safely (no div-by-zero throw)
- `fieldOf()` helper accepts both `object` and `array` chunk shapes — KbSearchService emits stdClass, fixtures use arrays.
